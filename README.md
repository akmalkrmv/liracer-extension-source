# LiRacer - Lichess Puzzle Tracker (Source Code)

This repository contains the **source code** for the **LiRacer** Chrome extension.

> **Note**: For the installable version and build files, please visit the main repository at:
> [**https://github.com/akmalkrmv/liracer-extension**](https://github.com/akmalkrmv/liracer-extension)

The repository linked above serves as a **build folder** for convenience and easier distribution. If you want to contribute, modify, or understand how the extension works, you are in the right place.

---

### Folder Structure
- `src/background`: Logic that runs in the background service worker.
- `src/content`: Scripts that inject into Lichess pages to track puzzle activity.
- `src/popup`: The Angular application for the extension's popup window.
- `scripts/`: Custom build and watch scripts.
